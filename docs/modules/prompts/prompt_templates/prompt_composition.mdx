# 提示组合

流水线提示模板允许您将多个单独的提示模板组合在一起。
当您想重用单个提示的部分时，这可能很有用。

与将`inputVariables`作为参数不同，流水线提示模板需要两个新参数:

- `pipelinePrompts`: 包含字符串(`name`)和以`promptTemplate`传递的`PromptTemplate`的对象的数组。
  Each `PromptTemplate` will be formatted and then passed to the next prompt template in the pipeline as an input variable with the same name as `name`.

- `finalPrompt`: 将返回的最终提示。
